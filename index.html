What Discord uses for formatting

Discord uses a modified version of Markdown. It adds and removes components of Markdown.

But how does Markdown work?

It's easy. Markdown is like an easier-to-use HTML. It is watered down, meaning it has less features, but a modified version, Markdown Extra, adds a few more, including the ability to use Markdown in HTML!
